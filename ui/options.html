💾 全局配置备份与恢复

此处支持统一导出或导入 AIO 插件的所有功能配置。建议在更换设备或重装系统前进行备份。

导出备份

将当前所有功能的设置项保存为 JSON 文件到您的本地电脑。

从备份恢复

选择之前导出的 JSON 文件以恢复配置。注意：这将覆盖所有当前设置！

危险操作

将所有设置恢复为出厂默认状态。此操作不可逆！
